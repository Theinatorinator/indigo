# QHY CCD driver

http://www.qhyccd.com

## Supported devices

All QHY cameras.

This driver supports hot-plug (multiple devices).

## Supported platforms

This driver depends on 3rd party library and is supported on Linux (Intel 32/64 bit and ARM v6+) and MacOS.
