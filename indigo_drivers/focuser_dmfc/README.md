# PegasusAstro DMFC focuser driver

http://www.pegasusastro.com

## Supported devices
* PegasusAstro Motor Focus Kit
* PegasusAstro FocusCube

Single device is present on startup (no hot-plug support).

## Supported platforms

This driver is platform independent.
