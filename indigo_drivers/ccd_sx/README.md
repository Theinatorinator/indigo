# StarlightXpress CCD driver

http://www.sxccd.com

## Supported devices

SXVF-Mxxx
SXVR-Hxxx
Trius-SXxxx
Lodestar Autoguider
Superstar Autoguider
CoStar Autoguider
Ultrastar Autoguider

This driver supports hot-plug (multiple devices).

## Supported platforms

This driver is platform independent.
