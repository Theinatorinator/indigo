# StarlightXpress wheel driver

http://www.sxccd.com

## Supported devices

SX USB filter wheel
Mini filter wheel with OAG
Maxi Filter Wheel

This driver supports hot-plug (single devices).

## Supported platforms

This driver is platform independent.
