# Deep Sky Dad AF1/AF2 focuser driver

https://deepskydad.com

## Supported devices

AF1 and AF2 focusers.

This driver does not support hotplug (single device).

## Supported platforms

This driver is supported on Linux (Intel 32/64 bit and ARM v6+) and MacOS.
