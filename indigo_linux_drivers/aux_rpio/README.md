# Raspberry Pi GPIO AUX Controller driver

https://www.raspberrypi.org

## Supported devices

GPIO port of the Raspberry Pi SBCs.

No hot plug support. One GPIO device is present at startup.

## Supported platforms

This driver is supported on Linux (ARM v6+).
